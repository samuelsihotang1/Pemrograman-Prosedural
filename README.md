# [Pemrograman Prosedural]-Programming-Tasks
In the second semester, Del Institute of Technology Information Systems students delve into Procedural Programming with the C language. This course serves as a starting point for those new to programming and emphasizes the development of logic skills through hands-on problem-solving exercises. Throughout the course, students will learn to construct code using a series of sequential commands.
